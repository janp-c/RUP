Clase de producto de trabajo: Solución
Estos productos de trabajo se componen de aquellos que forman parte del producto o solución global, o que contribuyen directamente con la solución o producto.

Descripción principal

Muchos productos de trabajo contribuyen con la solución o producto global. Para validar la solución es necesario realizar pruebas y contar con datos de prueba. Los materiales de soporte del usuario de todos los tipos son necesarios en el producto final. Para compilar el producto final son necesarios el código fuente y otros elementos de implementación. Estos productos de trabajo, incluidos los que se entregan con el producto, se categorizan como solución.
